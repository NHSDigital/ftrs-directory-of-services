# FtRS Non‑Functional Requirements

This directory contains the FtRS NFR views and supporting programme governance.

- Programme‑level: cross‑cutting governance, approvals, and readiness
  - Generated Programme service pages:
    - [Programme – Service Index](nfr-by-service/programme/index.md)
    - [Programme – Governance controls](nfr-by-service/programme/governance.md)
  - Deprecated (manual): [Programme NFRs](programme-nfrs.md), [Programme NFR Checklist](checklists/programme-nfr-checklist.md)
- Service view: NFRs grouped by service
  - [NFR – By Service](nfr-by-service.md)
- Domain view: NFRs grouped by domain
  - [NFR – By Domain](nfr-by-domain.md)

Tip: programme‑level obligations set the minimum. Service and domain pages may set stricter thresholds; weaker thresholds require an explicit, traceable exception.
