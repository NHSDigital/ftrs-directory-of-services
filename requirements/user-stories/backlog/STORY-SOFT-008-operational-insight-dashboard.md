## Description

Create a unified operational dashboard exposing key service health metrics across uptime, latency, error rate, deployment frequency, and cost trends. The dashboard should update at least hourly and be accessible to stakeholders for monthly reviews to drive improvement actions.

## Acceptance Criteria

1. Dashboard includes uptime, latency, error rate, deployment frequency, and cost trends
2. Metrics update at least hourly with visible last-refresh timestamp
3. Monthly stakeholder review produces a recorded set of improvement actions linked to dashboard insights

## Out of Scope

- Predictive analytics for future incidents

- Evidence: dashboard screenshot, data refresh logs, and action item list
